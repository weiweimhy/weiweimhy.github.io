GoldenDict 使用指南

1223 字
2024-03-06
阅读时长: 3 分钟

简介

GoldenDict 是一款功能强大的开源词典软件，支持多种词典格式，可以集成在线翻译服务，是学习和工作中不可或缺的翻译工具。

主要特点：

  • 支持多种词典格式（StarDict、Babylon、Lingvo 等）
  • 可集成在线翻译服务（Google、DeepL 等）
  • 支持全文搜索
  • 跨平台支持（Windows、Linux、macOS）
  • 可定制的界面和快捷键

安装

使用 Scoop 安装

1
2

# 安装 GoldenDict
scoop install goldendict

便携模式说明

Scoop 安装的 GoldenDict 默认启用便携模式（Portable Mode），这意味着：

  1. 配置目录

    • 在程序目录下会自动创建 portable 文件夹
    • 所有配置文件都存储在这个文件夹中
    • 路径通常为：%SCOOP%/apps/goldendict/current/portable

  2. 词典目录限制

    • 便携模式下只能识别 content 文件夹中的词典
    • 无法通过界面添加其他目录的词典
    • 路径为：%SCOOP%/apps/goldendict/current/content

  3. 解决方案

    • 将词典文件复制到 content 文件夹中

    • 或者创建符号链接指向其他词典目录：

      1
2

      # 以管理员身份运行
mklink /D "%SCOOP%\apps\goldendict\current\content\词典名称" "实际词典目录路径"

手动安装

  1. 访问 GoldenDict 官网 或 GitHub 发布页
  2. 下载最新版本安装包
  3. 运行安装程序

词典推荐

词典下载

推荐网站：词典网站

将词典文件放到 %SCOOP%/apps/goldendict/current/content 路径

1. 英语词典

  1. 离线词典

    • 朗文当代英语词典（Longman）：英英释义详尽，例句地道
    • 牛津高阶英汉双解词典（Oxford）：权威性强，释义准确
    • 柯林斯高阶英汉双解词典（Collins）：例句丰富，搭配详细
    • 剑桥高阶英汉双解词典（Cambridge）：英美发音齐全
    • 韦氏高阶英汉双解词典（Merriam-Webster）：美式英语为主
    • Vocabulary.com Dictionary：词源解释详细，助记功能强

  2. IT 专业词典

    • 计算机英语词典：收录专业术语
    • 信息技术词典：包含最新 IT 领域词汇
    • 编程语言词典：各种编程语言的专业术语
    • GitHub Glossary：开源社区常用术语

2. 词典格式说明

GoldenDict 支持以下主流词典格式：

  1. Babylon (BGL)

    • 文件扩展名：.bgl
    • 特点：压缩率高，加载速度快

  2. StarDict

    • 文件扩展名：.ifo, .dict, .idx
    • 特点：开源格式，兼容性好

  3. Lingvo (DSL)

    • 文件扩展名：.dsl, .dsl.dz
    • 特点：支持复杂格式，显示效果好

  4. MDict

    • 文件扩展名：.mdx, .mdd
    • 特点：广泛使用，资源丰富

3. 词典下载地址

  1. 开源词典

    • ECDICT：英汉词典数据库
    • FreeDictionaryProject：自由词典项目

  2. 专业词典

    • IT-Dictionary：IT 专业词典
    • MDX/MDD 词典下载：胡正维护的词典下载站

在线翻译配置

1. Google 翻译

  1. 安装翻译工具

    1
2
3
4
5

    # 安装 Python（如果未安装）
scoop install python

# 安装 Google 翻译工具
pip3 install google-translate-for-goldendict

  2. 配置 GoldenDict

    • 编辑 > 词典 > 词典来源 > 程式
    • 点击 添加
    • 设置以下参数：
      • 类型：Html
      • 名称：Google Translate
      • 命令行：python -m googletranslate zh-CN %GDWORD%
      • 图示：可选，下载 google_translate.png
