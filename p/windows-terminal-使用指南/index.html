Windows Terminal 使用指南

880 字
2024-03-06
阅读时长: 2 分钟

简介

Windows Terminal 是一个现代化、多标签的终端应用程序，专为使用命令行工具设计。它提供了许多强大且灵活的功能，可以替代经典的命令提示符和 PowerShell 窗口。

主要特性：

  • 多标签页支持
  • 分屏功能
  • 自定义主题
  • 丰富的快捷键
  • 支持多种 Shell（PowerShell、CMD、WSL 等）

  1. Microsoft Store

    • 直接安装
    • 优点：自动更新，安装简单

  2. GitHub 发布页

# 开启代理
function proxy-on {
    param($proxy = "http://127.0.0.1:1080")
    $env:http_proxy = $proxy
    $env:https_proxy = $proxy
    Write-Host "代理已开启: $proxy"
}

# 关闭代理
function proxy-off {
    $env:http_proxy = ""
    $env:https_proxy = ""
    Write-Host "代理已关闭"
}

# 设置环境变量
function set-env {
    param(
        [string]$key,
        [string]$value,
        [string]$scope = "u"  # u: 用户, m: 系统
    )
    
    if ($scope -eq "u") {
        [System.Environment]::SetEnvironmentVariable($key, $value, "User")
    } else {
        [System.Environment]::SetEnvironmentVariable($key, $value, "Machine")
    }
    Write-Host "环境变量已设置: $key = $value"
}
